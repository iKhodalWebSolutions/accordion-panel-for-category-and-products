Beautiful woocommerce product accordion that displays category wise product listing accordion tabs and filter. 

== Description ==

Category and Products Accordion Panel is the best plugin for the Home and Sup-Pages that will enhance the shopping experience and view the products more categorized with accordion panel for woocommerce in wordpress.

Customer can search and view the products according to categories accordion pane. This plugin supports unlimited widgets for the sidebar. It supports fully responsive layout with four different styles.

== Installation ==

1. Upload and extract the zip file downloaded to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress 
3. Navigate to 'WCAP' page from dashboard and create new shortcode to add into any page content (Premium/Pro version only)

== Frequently asked questions == 

= Can we override or change designing templates and style?  =

Yes, There is dynamic text and background color selection from admin panel.

= How to create shortcode and use direct in page or posts body content?  =

Premium version allows to configure shortcodes and can be created from left dashboard menu called "WCAP".

= Can I use multiple widgets on the same page?  =

Yes, Supports multiple widgets to display category accordion panel.
